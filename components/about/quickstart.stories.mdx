## 本地调试

> 使用 npm link @wont/react-ui`和`npm unlink @wont/react-ui`

```bash
## 到@wont/react-ui工程下执行
npm link

## 到业务工程下执行
npm link @wont/react-ui

## 取消本地调试执行
npm unlink @wont/react-ui
```

## 使用步骤

### 1. npm install

```bash
npm i @wont/react-ui -S
```

### 2. [babel-plugin-component](https://www.npmjs.com/package/babel-plugin-component)

#### install

```bash
npm i babel-plugin-import -D
```

#### .babelrc.js

```js
module.exports = {
    plugins: [
        [
            'import',
            {
                libraryName: '@wont/react-ui',
                libraryDirectory: 'es',
                camel2DashComponentName: false,
                style: (name) => {
                    console.log('name :>> ', name)
                    return `${name}/index.less`
                },
            },
        ],
    ],
}
```

or

```bash
npm i babel-plugin-component -D
```

#### .babelrc.js

```js
"plugins": [
  [
    "component", {
      "libraryName": "@wont/react-ui",
      "libDir": "lib",
      "style": "index.css"
    }
  ]
]

```

#### 与 xx 同时使用

```js
"plugins": [
  [
    "component", [
        {
            "libraryName": "xx",
            "style": true
        },
        {
            "libraryName": "@wont/react-ui",
            "libDir": "lib",
            "style": "index.css"
        }
    ]
  ]
]
```
